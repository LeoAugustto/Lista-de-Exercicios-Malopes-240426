Projeto em React JS contendo os exercícios 1 ao 8 resolvidos em uma única aplicação.

## Como executar

1. Abra o terminal na pasta do projeto.
2. Instale as dependências:

```bash
npm install
```

3. Execute o projeto:

```bash
npm run dev
```

4. Abra o endereço informado no terminal, normalmente:

```text
http://localhost:5173
```

## Organização

Cada exercício está separado em um componente dentro da pasta:

```text
src/exercicios/
```

Arquivos principais:

- `src/App.jsx`: menu principal e seleção dos exercícios.
- `src/exercicios/Ex1.jsx`: mostra 10 strings em tamanhos diferentes.
- `src/exercicios/Ex2.jsx`: gera tabela com n linhas e texto digitado.
- `src/exercicios/Ex3.jsx`: valida formulário com nome, endereço, telefone e email.
- `src/exercicios/Ex4.jsx`: calcula o n-ésimo elemento de Fibonacci.
- `src/exercicios/Ex5.jsx`: mostra os n elementos da sequência de Fibonacci em uma tabela.
- `src/exercicios/Ex6.jsx`: mostra uma tabela 4x4 de números inteiros aleatórios.
- `src/exercicios/Ex7.jsx`: calculadora básica com dois valores.
- `src/exercicios/Ex8.jsx`: calculadora no modelo visual solicitado.
